Definición

¿Qué es Git?
Git es un software de control de versiones diseñadas por Linus Torvalds, pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando tienen un gran número de archivos de código fuente.
¿Qué es un repositorio Git?
Un repositorio Git es un almacenamiento virtual de proyectos. Le permite guardar versiones de su código, a las que puede acceder cuando sea necesario.

Comandos Básicos

  • git config: Este comando establece el nombre del autor y la dirección de correo electrónico, respectivamente, para usar con sus confirmaciones.
  • git init: Este comando se usa para iniciar un nuevo repositorio.
  • git clone:Este comando se utiliza para obtener un repositorio de una URL existente.
  • git add: Este comando agrega un archivo al área de "stagging".
  • git commit: Este comando registra o captura instantáneamente el archivo de forma permanente en el historial de versiones.
